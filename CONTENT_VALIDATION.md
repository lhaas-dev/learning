# CONTENT VALIDATION – MVP

## 1. Ziel

Sicherstellen, dass nur:
- prüfbare
- eindeutige
- didaktisch sinnvolle Inhalte

im System landen.

---

## 2. Automatische Validierung

- Quality Filter Prompt (PROMPTS.md)
- Regeln:
  - eine Idee pro Check
  - kurze Antwort
  - keine Vagheit
  - realistische Prüfungsfrage

---

## 3. Manuelle Kontrolle (User)

User kann:
- Checks editieren
- löschen
- Concepts deaktivieren

---

## 4. Speicherung

Nur Inhalte mit:
- KEEP oder EDIT
werden gespeichert.

DROP → wird verworfen.

---

## 5. Feedback Loop (später)

Aggregierte Daten:
- häufig falsch beantwortete Checks
- ungewöhnlich hohe Fehlerquote
→ Kandidaten für Überarbeitung
